# VUE-IONIC

> ionic ui 的 vue 实现版本
- [Vue官网](http://vuejs.org/)  
- [Ionic官网](http://ionicframework.com/)    

## 启动     

``` bash
# install dependencies
npm install

# 开发 浏览器中查看 http://localhost:8080
npm run dev

# 发布
npm run build

```
> 通常说 可能你按上面执行呢 并木有跑起来, 请联系我
